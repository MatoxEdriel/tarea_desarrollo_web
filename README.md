# MVC-PHP

Proyecto web desarrollado con el patrón de arquitectura **MVC (Modelo - Vista - Controlador)** usando **PHP** y **HTML**, ejecutado en **XAMPP** y con base de datos en **MySQL (phpMyAdmin)**.

El sistema gestiona información relacionada con velas, como categorías y productos.

---

## Requisitos

- XAMPP (Apache y MySQL)
- Navegador web (Chrome, Edge, Firefox, etc.)
- Editor de código (VS Code, Sublime, etc.) — opcional

---

## Instalación

1. Instala y abre **XAMPP**.
2. Inicia los servicios:
   - Apache
   - MySQL
3. Copia la carpeta del proyecto en:

C:\xampp\htdocs\MVC-PHP

4. Abre **phpMyAdmin** desde:

http://localhost/phpmyadmin

5. Crea la base de datos con el nombre:

mcv_candles

6. Importa el archivo SQL de la base de datos (si tienes uno).
7. Configura la conexión a la base de datos en el archivo de conexión del proyecto.

---

## Uso

1. Abre tu navegador.
2. Ve a:

http://localhost/MVC-PHP

3. El sistema debería cargar la página principal del proyecto.

---

## Estructura del proyecto (MVC)

- **model/** → Lógica de datos y acceso a la base de datos
- **view/** → Vistas (HTML, interfaces)
- **controller/** → Controladores (lógica de la aplicación)
- **index.php** → Punto de entrada del sistema
